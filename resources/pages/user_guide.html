5. Error Handling

Error Code Meaning Resolution
0001 This error alerts us that the application has crashed. This is usually due to lack of processing or lack of resources. Try closing non needed applications and stop background updates
0002 This error is for when Gathering the Magik cannot connect to the network. Attempt standard reconnecting techniques and restart the application.
0003 This error is for when the user has been banned from a match and continues attempting to join. Get unbanned or stop trying to join.
0005 This error is received when the User has been deemed mentally defective. User must provide a doctor’s note authorizing further use of our application.
0006 This error is for any failure that cannot be classified by the other categories. Try completely closing the Gathering the Magik application, and reopening it.
0007 This error is for testing purposes. Restart the whole system.
0008 This error means that our application is having “technical” issues. Please contact us.
0009 This error is for when the user has been banned from using our application yet attempts to use it continue. Uninstall forever.
0010 This error is given when the application detects long durations of time between restroom breaks. Relieve yourself.
0666 Occurs when the device on which our application is installed becomes possessed by an evil spirit. Please see a priest.
5318008 This error was programmed by a rogue and immature software developer, and keeps showing back up. Disregard, we will handle this.
