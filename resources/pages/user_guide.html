3. Description of System Functions

3.1 - Functions and Their Uses

  • Creating a Profile
    • A user profile is necessary for confirming accounts and assisting with connecting with other users.
  • Searching for Games
    • Allows the user to find a suitable game to join.
  • Hosting Games
    • Allows the user to create games in order for others to join.
  • Profile Customization
    • Modify settings to better tailor Gathering the Magik toward the user.
  • Checking News
    • See what is going on with Gathering the Magik and related events.
  • Becoming a Premium Member
    • Take advantage of our paid service and it’s features.

3.2 - Detailed Description of Functions

  • Creating a Profile
    • Upon entering the application there is a screen which prompts the user with two options.
    • Either they have an account already, or will have to make one.
    • Once create an account is selected, there will be a prompt to enter an email that the application will be tied to.
    • A password will be sent to the given email address that the user can then use to view and/or edit their profile.
    • The user will be given a randomly generated name by default.
  • Searching for Games
    • When the “join” option is selected, the user will begin the process of joining a game.
    • The user will be presented with several options:
      • Scroll through games in their area.
      • Refine search with a keyword for games in their area.
      • Choose a genre of games to refine search with.
    • Posted alongside the game will be its capacity and occupancy.
    • Once a game has been selected you can see the names of the host and those attending.
    • Selecting the large “JOIN” button will add you to the list of occupants.
    • At this point, the user will be expected to participate in said event.
  • Hosting Games
    • When the “host” option is selected, the user will begin the process of hosting a game.
    • A list of details will need to be provided so that other users can find the game:
      • The name of the game that you will be playing.
      • The amount of players allowed.
      • The location.
      • The time and date desired.
      • The estimated duration.
      • Any additional details required.
    • Selecting the “HOST” option will create the event with the user as the host.
    • Once the prior processes are finished, the game will become available for others to join, and the names of those attending will be viewable.
    • At this point, the user will be expected to host said event.
  • Profile Customization
    • Go into the options on the home page.
    • There are sub options which allow for customization of the user account:
      • Change password.
        • Allows the input of a new password for logging in to the profile.
      • Blacklist.
        • Allows the user to add other users to a list. This will block listed names from joining the user’s hosted games.
      • Adjust the size of games that count as in “In your Area.”
      • Display.
        • Changes contrast and brightness of the application interface.
  • Checking News
    • Under “News.”
    • The “In the Media” tab will inform the user about upcoming events, as well as deliver news about games that match the users interest.
    • “Updates” tab shows recent updates to application and future plans.
  • Becoming a Premium Member
    • Gathering the Magik offers the option to upgrade to a Premium User.
    • Premium users receive access to additional features including, but not limited to:
      • Changing a username to a new, easier to read username from a previously generated list.
      • Gain the ability to rate other user’s abilities.
      • Report other players.
      • Receive discounted updates.

3.3 - Preparation of Function Inputs

  • Creating a Profile
    • Provide an email to link with the account.
  • Searching for Games
    • Modify what is considered the “nearby area” under the profile customization to get better search results.
  • Hosting Games
    • Add tags to the description; often these are helpful for people trying to get better details when looking for a game to join.
  • Profile Customization
    • Input the old password once in order to create a new password.
  • Checking News
    • Entering user interests in order to provide the most relevant news. Over time, interests will be adjusted by the way the user interacts with the application.
  • Becoming a Premium Member
    • Provide a method of payment and information for verification.

3.4 - Results

  • Creating a Profile
    • The user is now registered and can can continue using this account to get connected with other players.
  • Searching for Games
    • The user has browsed around and may have committed to joining a game.
  • Hosting Games
    • The user has agreed to host an event in their area.
  • Profile Customization
    • Settings are now modified to better suit the user’s preferences.
  • Checking News
    • The user will now be informed on recent and future updates, as well as upcoming events in the local area.
  • Becoming a Premium Member
    • User’s payment has been verified and they have been given access to all Premium Member features.

4. Operating Instructions

    This section provides detailed, step-by-step operation instructions for the system but does not describe the use of the functions contained within the system.

4.1 - Initiate Operation

    This application can be downloaded and installed through the default application marketplace on the user’s device.

    This section contains procedures for system log-in and initialization to a common known point, such as a system main menu screen. This initialization procedure should describe how to establish necessary modes of operation and set any initial parameters required for operation. Software installation procedures are to be included if the software is distributed on diskette and should be downloaded before each use.

4.2 - Maintain Operation

    The application will require constant network connection in order to perform automatic updates, authentication, and connection with surrounding gamers. These updates can be expected to have no more than a month between each other.

    If an error occurs during the update process, attempt to reload the page or run the application updater on a desktop PC.

    If on a mobile device, reinstall the application or wait for another update as one may be available once issues have been found in order to remedy any issues found in common systems.

4.3 - Terminate and Restart Operations

    If improper termination or stalling of the application has occurred, there are some ways the user can attempt to regain control or force restart the system.

    If using our desktop application, the user can type:

UP, UP, DOWN, DOWN, LEFT, RIGHT, LEFT, RIGHT, B, A, ENTER

This will allow the application to attempt to automatically find a resolution to the problem.

    If on a mobile device, the user should restart the entire machine by using the manufacturer defined key sequence.
